# Inventor 电子签名插件

用于 Autodesk Inventor Professional 2021 的专业电子签名插件。

## 功能特性

- 🔐 工程图电子签名生成
- ✅ 签名验证和有效性检查
- 👤 用户身份认证系统
- 🎯 Inventor 功能区集成
- 🔒 防篡改文档保护

## 系统要求

- Autodesk Inventor Professional 2021
- .NET Framework 4.7.2 或 .NET 6.0
- Windows 10/11

## 安装方法

1. 克隆此仓库
2. 使用 Visual Studio 2019+ 编译解决方案
3. 将插件部署到 Inventor
